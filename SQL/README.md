# SQL Learning Repository

This repository follows a structured SQL learning path from beginner to advanced level.

## Learning Roadmap

### 🟢 Beginner Level
- Introduction to SQL
- Setup Environment
- SELECT queries
- DDL commands
- DML commands

### 🟡 Intermediate Level
- Filtering data (WHERE)
- JOINs (basic & advanced)
- Set operators
- SQL functions (string, numeric, date/time)
- NULL handling
- CASE statements
- Aggregate functions
- Window functions (basics, aggregates, ranking, values)

### 🔴 Advanced Level
- Advanced SQL techniques
- Subqueries
- Common Table Expressions (CTE)
- Views
- Temporary tables & CTAS
- Execution plans
- Indexes & performance tuning
- Stored procedures & triggers
- Partitioning
- Optimization techniques
- AI + SQL concepts

### 🧪 Projects
- SQL Data Warehouse (Bronze / Silver / Gold layers)
- Exploratory Data Analysis (EDA)
- Advanced Data Analytics project

## Purpose
This repo is used to practice SQL step by step and build strong database and data analysis skills.
